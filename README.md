# Tic-Tac-Toe
A simple Tic-Tac-Toe program written in python:

[+]Computer is X and user is O
[+]Computer always starts 1st on the 5 square (in the middle of the board)
[+]If user wins -----> "Ai castigat!"("You winned!")
[+]If computer wins -----> "Am castigat!"("I winnned!")
[+]If draw -----> "Egalitate!" ("Draw")
